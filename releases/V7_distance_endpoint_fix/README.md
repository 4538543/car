# V7：修正编码器计数口径与固定距离停车

生成日期：2026-07-23

## 本版改动

- 第一题目标距离从 1050 mm 改为 1000 mm。
- 第三题 A→C 目标距离为 1269 mm。
- GPIO 当前统计 A 相上升沿，原始约 260 计数/圈；距离模块统一换算成四倍频等效计数，再与 1040 计数/圈匹配。
- 修复把 A 相单边沿计数直接除以 1040，造成距离被低估的问题。
- 第一题在约 750 mm 开始减速，到 1000 mm 时即使没有黑线也会制动停车。
- 第三题按三级速度行驶，到 1269 mm 时即使没有黑线也会制动并停在 C 点。
- 两路灰度黑线仍可在硬距离终点前触发停车。
- 第二题不加入距离减速或硬距离停车。

- `car_V7_firmware.out`：已通过 TI ARM Clang 编译链接的固件。
