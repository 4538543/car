# V6：增强末段减速与反向制动

生成日期：2026-07-23

## 本版改动

- 反向制动从 80 ms、220‰ 增加到 100 ms、280‰。
- 第一题最后约 250 mm 的速度从 210‰ 降至 160‰。
- 第三题剩余约 300 mm 时的速度从 290‰ 降至 260‰。
- 第三题剩余约 120 mm 时的速度从 210‰ 降至 160‰。
- 减速距离、目标距离和编码器 1040 计数/圈保持不变。
- 第二题仍不启用提前减速。
