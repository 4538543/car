# V3：编码器距离分段减速与独立按键防误触

生成日期：2026-07-23

## 本版改动

- A18、PB22、PB23、PB24 均确认配置为上拉输入、按下低电平。
- 每个按钮必须独立检测到松开高电平 100 ms 后才能触发，防止上电误启动。
- 任意一路按钮异常不会锁住其他按钮。
- 使用 65 mm 轮径和 250 脉冲/圈，将编码器平均计数换算为实际毫米距离。
- 第一题在 1050 mm 目标距离剩余约 250 mm 时，从 360‰ 降至 210‰。
- 第二题不加入提前减速，保持原控制方式。
- 第三题对角线暂按 1281 mm，采用三级速度：
  - 前段：360‰；
  - 剩余约 300 mm：290‰；
  - 剩余约 120 mm：210‰。
- 最终仍由至少两路灰度连续确认黑线 20 ms 后停车。
- 反向制动降低为 80 ms、220‰，减少轮胎打滑。

## 关键标定参数

位于 `app_config.h`：

- `WHEEL_DIAMETER_MM = 65`
- `ENCODER_PULSES_PER_REV = 250`
- `Q1_MIN_DISTANCE_MM = 1050`
- `Q3_DIAGONAL_DISTANCE_MM = 1281`
- `Q3_STAGE2_REMAINING_MM = 300`
- `Q3_STAGE3_REMAINING_MM = 120`

如果实测编码器每圈脉冲不是 250，只修改
`ENCODER_PULSES_PER_REV`，其他距离参数不需要重新换算成脉冲。
