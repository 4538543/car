# V8：第三题转弯快速稳定

生成日期：2026-07-23

## 本版改动

- 第三题转弯改为远近两段速度控制。
- 距目标超过 10°时，最大转向速度保持 260‰。
- 进入目标 ±10°后，最大转向速度限制为 130‰。
- 接近目标时的最低转向速度从 115‰ 降为 65‰，减少反复越过目标。
- 到位容差从 ±2°放宽到 ±3°。
- 稳定确认时间从 180 ms 缩短到 80 ms。
- 第三题转弯参数位于独立模块，不改变第一、二题直线 PID。

## 文件

- `car_V8_source.zip`：本版工程源码。
- `car_V8_firmware.out`：已通过 TI ARM Clang 编译链接的固件。
