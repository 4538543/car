# V2：四个启动按键独立触发

生成日期：2026-07-23

## 本版改动

- 删除四个按键必须同时释放 100 ms 才能全局解锁的逻辑。
- A18、PB22、PB23、PB24 分别进行独立的 20 ms 软件消抖。
- 任意一路按键或接线异常不再阻塞其他三个按键。
- 在 LCD 初始化期间已经按住的按钮，会在任务循环启动并持续检测 20 ms 后正常触发。
- 按住只触发一次，松开后才能再次触发，避免长按重复启动。
- 采用实车验证后的反向制动参数：200 ms、450‰。

## 按键有效电平

- A18：上拉输入，按下为低电平。
- PB22、PB23、PB24：上拉输入，按下为低电平。
